Cómo añadir mods al juego

¿Qué son los mods?

Los mods son modificaciones que se pueden realizar al juego Minecraft para añadir nuevas características, objetos, mobs, etc. Pueden ser creados por los propios jugadores o por desarrolladores externos.

Cómo instalar mods

Para instalar mods en Minecraft, necesitas tener una versión de Minecraft Java. Hay dos formas principales de instalar mods:

  • Usando Forge: Forge es una API que permite a los mods funcionar en Minecraft. Para instalar Forge, descarga el archivo de instalación de la página web de Forge y ejecútalo. A continuación, abre Minecraft y crea un nuevo mundo con el perfil "Forge".

  • Usando Fabric: Fabric es otra API que permite a los mods funcionar en Minecraft. Para instalar Fabric, descarga el archivo de instalación de la página web de Fabric y ejecútalo. A continuación, abre Minecraft y crea un nuevo mundo con el perfil "Fabric".

Una vez que hayas instalado Forge o Fabric, puedes descargar mods de sitios web como MinecraftMods.net o CurseForge. Para instalar un mod, descarga el archivo del mod y colócalo en la carpeta "mods" de tu instalación de Minecraft.

Una vez que hayas instalado los mods, puedes activarlos o desactivarlos en el menú "Mods" de Minecraft.
